막(Act) 선택
여정의 시작점 또는 다음 목적지를 선택합니다.
스크린샷 플레이스홀더
월드 맵 또는 막 목록. 잠긴 막은 회색 처리. 선택된 막은 상세 정보(보스 미리보기, 환경 타입)와 함께 강조.
흐름도
캐릭터 선택 -> 막 선택 -> 지도 생성 -> 지도 씬

필요 에셋

아트: 막 썸네일/아이콘 UI: 막 정보 패널 (이름, 설명, 난이도) SFX: 지도 펼침/선택 사운드

탐험 흐름

게임은 여러 막(Act)으로 나뉩니다. 각 막은 고유한 테마, 적 세트, 최종 보스를 가집니다.

  • 1막: 각성 (튜토리얼/기본 메카닉)
  • 2막: 심층 다이브 (심화된 적)
  • 3막: 코어 (최종 도전)
