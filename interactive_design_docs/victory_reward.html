전리품을 획득합니다. 덱을 강화하기 위해 신중하게 선택하세요.
스크린샷 플레이스홀더
"VICTORY" 배너. 3장의 카드 선택지 표시. 하단에 골드/포션 보상 목록.
흐름도
전투 종료 -> 승리 화면 -> 카드 보상 선택 -> 지도로 복귀

필요 에셋

UI: 승리 배너 UI: 카드 보상 컨테이너 UI: 골드/아이템 아이콘 SFX: 승리 팡파레 SFX: 보상 획득음

보상 유형

  • 카드 보상: 무작위 카드 3장 중 1장을 선택하여 덱에 추가.
  • 골드: 상점에서 사용하는 화폐.
  • 포션: 전투 중 사용하는 소모품.
  • 유물 (정예/보스): 강력한 패시브 아티팩트.
