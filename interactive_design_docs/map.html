지도 탐험
육각형 그리드를 통해 경로를 선택합니다.
스크린샷 플레이스홀더
육각형 그리드 지도. 현재 노드에 플레이어 토큰. 연결된 노드 강조. 먼 노드는 전장의 안개 처리.
흐름도
노드 선택 -> 전투 / 이벤트 / 상점 / 휴식 -> 지도로 복귀 -> 다음 노드

필요 에셋

UI: 육각형 타일 (일반, 정예, 보스, 이벤트, 상점, 휴식) UI: 플레이어 토큰 UI: 연결선 VFX: 전장의 안개 SFX: 발자국/이동 소리

지도 시스템

구조: 절차적으로 생성된 육각형 노드 망.

규칙:

  • 플레이어는 인접한 전방 노드로만 이동할 수 있습니다.
  • 한 번 클리어한 노드는 다시 방문할 수 없습니다.
  • 지도는 보스 노드에서 끝납니다.

노드 타입:

  • 전투: 일반 전투 조우.
  • 정예: 더 어려운 전투, 더 좋은 보상.
  • 이벤트: 위험과 보상이 따르는 서사적 선택.
  • 상점: 재화를 소비하여 카드/유물 구매.
  • 휴식: 체력 회복 또는 카드 강화.
