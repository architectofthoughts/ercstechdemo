타이틀 화면
게임의 진입점. 톤앤매너와 분위기를 설정합니다.
스크린샷 플레이스홀더
애니메이션 배경(입자, 안개)이 적용된 고품질 게임 로고 렌더링. "Press Start" 텍스트 점멸.
흐름도
타이틀 -> 메인 메뉴

필요 에셋

BGM: 타이틀 테마 (우울한 SF) SFX: 시작 버튼 호버/클릭 아트: 게임 로고 아트: 배경 이미지/영상 UI: "아무 키나 누르세요" 텍스트 UI: 버전 번호

게임 개요

장르: 로그라이크 덱빌딩 카드 배틀

플랫폼: 웹 (PC/모바일 반응형)

핵심 컨셉: "피날레(운명적 한 방)"와 "메모리얼(기억)"을 테마로 한 전략 전투.

핵심 루프

  1. 준비: 캐릭터 선택, DNA, 덱 구성.
  2. 탐험: 헥사곤 맵 이동 (전투, 이벤트, 보스).
  3. 전투: 적의 의도(Intent) 시스템을 포함한 턴제 카드 전투.
  4. 성장: 보상, 유물, 카드 수집.
  5. 결말: 추출(승리) 또는 초기화(패배).
