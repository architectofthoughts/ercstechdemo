이벤트 조우
런의 방향을 바꿀 수 있는 서사적 선택.
스크린샷 플레이스홀더
좌/상단에 대형 일러스트. 우/하단에 스토리 텍스트와 선택지.
흐름도
이벤트 시작 -> 선택 -> 결과 (보상/페널티) -> 지도로 복귀

필요 에셋

아트: 이벤트 일러스트 (분위기 위주) UI: 선택지 버튼 SFX: 이벤트 환경음

이벤트 유형

  • 거래: 체력을 골드로, 또는 카드를 유물로 교환.
  • 전투: 높은 보상을 위한 선택적 강적 전투.
  • 스토리: 게임플레이 효과가 미미한 로어 확장.
  • 강화: 덱의 카드를 강화.
