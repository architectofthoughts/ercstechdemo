흐름도

필요 에셋

UI: 슬라이더 (볼륨) UI: 체크박스/토글 UI: 드롭다운 (해상도)

설정 항목

  • 오디오: 마스터, BGM, SFX, 음성 볼륨.
  • 그래픽: 해상도, 전체화면, 품질 (저/중/고), 수직동기화.
  • 게임플레이: 텍스트 속도, 화면 흔들림 (On/Off), 빠른 모드.
