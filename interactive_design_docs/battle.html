전투 시스템
핵심 게임플레이 루프. 전략, 자원 관리, 그리고 화려한 처형.
스크린샷 플레이스홀더
하단에 플레이어 손패. 중앙에 적 스프라이트. 적 머리 위에 의도(Intent) 아이콘. 마나/체력 UI.
흐름도
카드 드로우 -> 카드 사용 (마나 소모) -> 턴 종료 -> 적 턴 -> 반복 -> 승리/패배

필요 에셋

아트: 카드 프레임 (공격, 스킬, 파워) 아트: 적 스프라이트 (대기, 공격, 피격) UI: 의도 아이콘 (검, 방패, 디버프) VFX: 카드 사용 효과 (베기, 방어, 화염) VFX: 피날레 폭발 SFX: 카드 드로우, 셔플, 타격음

메카닉

턴 구조

  1. 드로우 페이즈: 카드 5장 드로우. 마나 초기화.
  2. 플레이어 턴: 카드를 사용하여 공격, 방어, 버프 수행.
  3. 적 턴: 적들이 예고된 행동(Intent)을 수행.

특수 시스템

  • 피날레: 적 체력 < 50% 이고 플레이어 마나/카드가 충분할 때, 위로 스와이프하여 처형기 발동.
  • 데스티니 드로우: 위기 상황에서 손패를 모두 버리고 강력한 "운명의 카드"를 드로우.
  • 물리 효과: 카드는 물리적 무게감을 가지며 시각적 재미를 위해 회전하거나 던질 수 있음.
