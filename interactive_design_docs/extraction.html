스크린샷 플레이스홀더
하이테크 인터페이스. "추출 점수(EP)" 카운터. EP를 "원시 에테르" 또는 "유산 카드"에 소비하는 옵션.
흐름도
막 클리어 / 승리 -> 추출 화면 -> EP 소비 -> 메인 메뉴로 복귀

필요 에셋

UI: 홀로그램 인터페이스 UI: 자원 아이콘 (EP, 에테르) VFX: 데이터 업로드 애니메이션 SFX: 디지털 처리음

추출 시스템

추출 점수 (EP): 처치한 적, 획득한 골드, 클리어한 막에 따라 획득합니다.

EP 소비:

  • 물질 변환: EP를 원시 에테르(가챠 재화)로 변환합니다.
  • 유산 카드: 현재 덱에서 카드 1장을 저장하여 다음 런의 시작 덱에 포함시킵니다.
  • 메모리 뱅크: 나중에 사용하기 위해 EP를 저장합니다.
