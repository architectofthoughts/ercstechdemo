캐릭터 선택
요원(Agent)을 선택하고, DNA를 설정하며, 시작 덱을 미리 봅니다.
스크린샷 플레이스홀더
중앙에 3D/2D 캐릭터 회전목마. 좌측에 스탯/소개. 우측에 DNA/덱 옵션.
흐름도
요원 선택 -> DNA 설정 -> 덱 미리보기 -> 확정 -> 막(Act) 선택

필요 에셋

아트: 캐릭터 전신 일러스트 UI: 스탯 아이콘 (체력, 마나 등) UI: DNA 나선 시각화 UI: 카드 미리보기 툴팁 SFX: 캐릭터 선택 음성

시스템 상세

캐릭터 선택

플레이어는 해금된 요원 중 하나를 선택합니다. 각 요원은 고유한 시작 스탯과 시그니처 카드를 가집니다.

DNA 시스템

패시브 특성을 시각적으로 표현합니다. 플레이어는 DNA 구조를 분석하고(향후 업데이트에서 수정 가능) 시작 보너스를 변경할 수 있습니다.

덱 미리보기

시작 덱 구성을 확인합니다. 런을 시작하기 전 전략적 계획을 세우는 데 중요합니다.
