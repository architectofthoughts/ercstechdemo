메인 메뉴
게임을 시작하고 메타 시스템에 접근하는 중앙 허브.
스크린샷 플레이스홀더
좌/우측에 메뉴 옵션 배치. 중앙에 캐릭터/배경 아트. 동적인 카메라 움직임.
흐름도
작전 시작 -> 캐릭터 선택
메모리얼 -> 메모리얼 앱
가챠 -> 가챠 씬
옵션 -> 설정 오버레이
종료 -> 데스크탑

필요 에셋

BGM: 메인 메뉴 테마 (차분함, 기대감) UI: 메뉴 버튼 (시작, 메모리얼, 가챠, 옵션, 종료) 아트: 동적 배경 SFX: 메뉴 이동/선택

디자인 언어

테마: SF + 판타지 (마법 공학)

색상: 딥 다크 블루/블랙 (배경), 매지컬 골드 (강조), 네온 핑크/블루 (효과)

타이포그래피: 가독성 좋은 고딕체 + 장식적인 헤드라인.

인터랙션

  • 마이크로 애니메이션: 버튼 호버 효과, 텍스트 타이핑 효과.
  • 전환: 메뉴 간 부드러운 페이드인 및 슬라이드 전환.
